# LHXC Show Submissions

This is a Wordpress plugin that allows users to submit their shows to the LHXC Show Submissions page. Each submission is then reviewed before any attached flyers are added to the Flyer Archive or show info added to the Shows Calendar.

## Installation
Download a ZIP file, unzip and either upload to your Wordpress site or drop it into your wp-content/plugins folder.

## To Build
Run `npm run build` to build the plugin.
